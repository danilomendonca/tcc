O aplicativo pode ser acessado via três URLs diferentes. Provavelmente a URL mais acessada é gerada quando o usuário clica na aba "Rede UnB", presente na página da UnB no Facebook. É provável que isto aconteça porque esta URL foi o principal canal de divulgação do aplicativo. Esta é a URL em questão: . A URL de visualização do aplicativo é http://apps.facebook.com/frame\_embutido".
A URL externa é "http://adsum.com.br/facebook/pedro3/". Esta última URL é a responsável por carregar o conteúdo do servidor de hospedagem para as outras URLs.
